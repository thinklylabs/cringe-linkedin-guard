Gemini API Key

To get your Gemini API key, follow these steps:

  1. Sign up or log in to your Google Cloud account.
  2. Create a new project or select an existing one.
  3. Enable the Vertex AI / Gemini API in the API Library.
  4. Generate a new API key and copy it.
  5. Paste the key in the field below and click "Save".
